To run the workspace
----

                Clone proj_5
                cd ..
                catkin_make
                source ./devel/setup.bash
                (open new tab)
                roscore
                (open new tab)
                roslaunch rrt_package rrtsimulationkobuki.launch 
                (open new tab)
                roslaunch rrt_exploration single.launch
